# 0.0.37 Toolbar Visual Harmonization

## Goal

Harmonize the revision graph toolbar so search controls and graph zoom controls use the same grouped visual treatment as the Git/graph operations toolbar.

## Scope

- Treat the search area as a toolbar action slot while preserving the existing search field, result count, previous, next, and clear controls.
- Group the main graph zoom out, reset, and zoom in controls inside the same slot style used by repository actions.
- Keep existing control IDs, titles, aria labels, keyboard behavior, and command wiring unchanged.

## Non-Goals

- Redesigning the toolbar layout beyond the three existing slots.
- Changing minimap zoom controls.
- Replacing the existing custom toolbar icons.
- Changing graph behavior, search matching, zoom levels, or repository actions.

## Acceptance Criteria

- Search, repository actions, and graph zoom controls read as visually related toolbar groups.
- Existing webview shell tests still find all search, action, and zoom controls.
- `npm run build` and focused webview tests pass.
