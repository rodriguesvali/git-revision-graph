# 0.0.37 Zoom Toolbar Icons

## Goal

Improve the visual quality of the main revision graph zoom controls.

## Scope

- Replace text glyphs for zoom out, reset zoom, and zoom in with inline SVG toolbar icons.
- Keep the icons aligned with the repository-action and search toolbar icon style.
- Preserve button IDs, titles, aria labels, keyboard shortcuts, zoom levels, and runtime wiring.

## Non-Goals

- Changing graph zoom behavior.
- Changing minimap zoom controls.
- Changing toolbar grouping or layout beyond icon replacement.

## Acceptance Criteria

- Main graph zoom controls render as consistent stroke icons.
- Existing webview shell tests still find all zoom controls.
- Focused webview tests and build pass.
