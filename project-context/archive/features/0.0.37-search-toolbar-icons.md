# 0.0.37 Search Toolbar Icons

## Goal

Improve the visual quality of the search navigation and clear controls in the revision graph toolbar.

## Scope

- Replace text glyphs for previous result, next result, and clear search with inline SVG toolbar icons.
- Keep the icons aligned with the existing repository-action toolbar icon style.
- Preserve button IDs, titles, aria labels, keyboard behavior, and search runtime wiring.

## Non-Goals

- Changing search matching behavior.
- Changing search result navigation.
- Changing toolbar grouping or layout beyond the icon replacement.

## Acceptance Criteria

- Search previous, next, and clear controls render as consistent stroke icons.
- Existing webview shell tests still find all search controls.
- Focused webview tests and build pass.
