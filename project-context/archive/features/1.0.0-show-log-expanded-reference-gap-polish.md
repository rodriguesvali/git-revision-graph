# 1.0.0 Show Log Expanded Reference Gap Polish

Build implementation complete for the `1.0.0` Show Log expanded reference gap polish.

## Goal

Remove the visual gap in Show Log expanded commit rows when the commit has reference badges or short-stat metadata above the changed-file list.

## Scope

- Keep the existing Show Log editor panel, compact graph, row layout, reference badges, and changed-file expansion behavior.
- Extend only the main compact-graph segment for expanded rows with visible metadata before the continuation rows begin.
- Preserve normal compact row density for commits without expanded metadata.

## Acceptance Criteria

- Expanded commits with refs or stats keep the compact graph line continuous through the metadata row.
- Expanded commits without metadata keep the existing compact graph spacing.
- Changed-file rows remain aligned with the message column.
- Existing Show Log shell tests pass.
