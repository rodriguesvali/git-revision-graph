# Release 0.0.17 Prioritization

## Intent

Version `0.0.17` should consolidate the recent graph and compare workflow improvements, with emphasis on fit-and-finish, lower-friction review flows, and stronger consistency across the extension surfaces.

## Proposed Focus Areas

- Graph ergonomics
  Continue refining graph-side interactions so common actions stay predictable and lightweight during day-to-day navigation.

- Compare review polish
  Improve the compare-results experience in ways that preserve the new flexibility while reducing friction for larger file reviews.

- Workbench consistency
  Tighten visual and behavioral consistency between the extension iconography, view titles, and other workbench-facing surfaces.

## Candidate Themes

- Small workflow wins around compare, fetch, sync, and other repeated review actions
- Consistency passes on iconography, labels, and surface-specific affordances
- Focused UX cleanup that improves density and discoverability without redesigning the graph architecture

## Out of Scope by Default

- Large graph host or rendering architecture changes
- Full parity work against the built-in Source Control graph
- Broad SCM workflow redesigns that are not directly justified by the current extension surfaces
