# Project Context Archive

Status: Historical
Created: 2026-06-30

This archive preserves completed or superseded AAMAD artifacts.

Archived material remains useful for traceability, but it is not current product or architecture scope unless explicitly reopened by the maintainer.

## Contents

- `features/`: completed feature and hotfix build artifacts.
- `releases/`: historical release readiness logs.
- `prioritization/`: release prioritization notes from earlier cycles.
- `define/`: superseded define-phase documents.
- `analysis/`: older analysis and opportunity notes.

## Reopening Rule

When historical work becomes active again, create a new concise artifact in the active `project-context/` tree and link back to the archived source. Do not edit the archived document as if it were live scope.
