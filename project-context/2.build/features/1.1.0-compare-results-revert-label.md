# Compare Results Revert to This

## Goal

Expose `Revert to this` consistently from Compare Results file context menus and make it replace the workspace copy with the selected compare-side copy.

## User Workflow

- Run a compare with the worktree or between two refs.
- Open a changed file row context menu in Compare Results.
- Choose `Revert to this` to write the selected revision's file state into the workspace.

## Scope

- Show `Revert to this` for file rows that have a restorable ref.
- For compare-with-worktree results, restore from the compared ref into the workspace.
- For ref-to-ref results, restore from the right/target ref into the workspace.
- Confirm before overwrite when the workspace copy already has local changes.

## Non-goals

- Do not add multi-file revert.
- Do not change diff opening or clipboard behavior.

## Acceptance Criteria

- Compare-with-worktree file context menus show `Revert to this`.
- Ref-to-ref compare file context menus also show `Revert to this`.
- `Revert to this` writes the selected compare-side copy into the workspace.
- A modal confirmation is shown when the workspace copy already has local changes.
- Existing compare result diff, copy, and message routing remains unchanged.
