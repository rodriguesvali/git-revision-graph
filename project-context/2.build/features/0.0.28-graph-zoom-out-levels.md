# 0.0.28 Graph Zoom-Out Levels

## Goal
Give users more zoom-out range in the main revision graph.

## User Workflow
Users inspect a dense or wide revision graph and need to zoom farther out without changing the existing maximum zoom-in behavior.

## Scope
- Keep the existing graph zoom levels.
- Add five additional levels on the zoom-out side.
- Preserve the existing toolbar `+` and `-` stepping behavior.

## Non-Goals
- Changing minimap zoom levels.
- Adding a user setting for zoom levels.
- Changing graph layout, node sizing, or viewport preservation behavior.

## Touched Surfaces
- `src/revisionGraph/webview/script/bootstrap.ts`
- `test/revisionGraphWebview.test.ts`

## Acceptance Criteria
- The main graph zoom list has 10 fixed levels.
- The previous levels `0.6`, `0.8`, `1`, `1.25`, and `1.5` remain available.
- The added levels are below `0.6`, increasing zoom-out range only.
- Toolbar zoom buttons continue to step through the fixed list.

## Test Plan
- `npm run build` passed.
- `npm test` passed with 239 tests.

## Manual Validation
- Open a large revision graph in the Extension Development Host.
- Click the graph toolbar `-` control repeatedly and verify the graph can zoom out beyond the previous `0.6` minimum.
- Click the graph toolbar `+` control and verify it returns through the same levels up to the existing `1.5` maximum.

## Release-Note Impact
- Added five more main graph zoom-out levels for large graph navigation.

## Handoff Notes
- The minimap zoom range is unchanged.
- Automated coverage now asserts the main graph zoom list includes 10 levels with the added levels below `0.6`.
