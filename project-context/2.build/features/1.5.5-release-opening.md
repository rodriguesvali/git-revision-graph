# 1.5.5 Release Opening

## Status

Release cycle opened on 2026-06-30. Feature implementation has not started.

## Goal

Open `1.5.5` as a focused graph-navigation patch after the maintainer-confirmed `1.5.4`
release and establish `Focus Descendants` as the first approved development slice.

## Context

`1.5.4` introduced Focus Range for a selected ordered pair. Follow-up product review identified
a complementary branch-heavy repository workflow: select one visible revision and inspect only
the references and graph paths that derive from it. Existing viewport virtualization limits DOM
mounting but does not remove full focused-scene projection and layout work, so this feature is
primarily a navigation improvement with a secondary layout-size benefit.

## Scope

- Apply the approved package metadata bump from `1.5.4` to `1.5.5`.
- Record `1.5.4` as the published baseline based on maintainer confirmation in this opening.
- Create the `1.5.5` Define, Build, prioritization, changelog, and release-readiness state.
- Approve `Focus Descendants` for implementation under its focused feature artifact.

## Non-Goals

- No feature implementation in the release-opening change.
- No VSIX packaging or Marketplace publication.
- No command, contribution point, setting, dependency, or product-surface change.
- No scope beyond `Focus Descendants` without a subsequent maintainer decision.

## Acceptance Criteria

- `package.json` and root package-lock metadata report `1.5.5`.
- AAMAD Define, Build, and Deliver artifacts identify `Focus Descendants` as the initial scope.
- `1.5.4` publication is recorded without inventing unavailable VSIX or Marketplace metadata.
- Packaging and publication remain explicit future approvals.
